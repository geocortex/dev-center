This article will walk you through creating a new workflow activity for web applications.

## Prerequisites

[Download and set up](sdk-web-overview.mdx#download) the Geocortex Workflow TypeScript SDK.

## Create an Activity

1. Open up a terminal shell in the SDK folder.
1. Run the command `npm run activity` and name the activity `MyCustomActivity`.

:::note
This script creates a custom activity skeleton that you can build on with your own custom logic.
:::

## Deploy the Activity

Follow the instructions to [build the activity pack](sdk-web-overview.mdx#building) and [deploy it](sdk-web-overview.mdx#deployment).

## Test the Activity

Once your activity pack is hosted and registered, your custom activity should appear in the activity toolbox in Geocortex Workflow Designer alongside the built-in activities, and can be used in the graphical interface like any other activity.

Now you can [build a workflow](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/get-started.htm#Get_Started%3FTocPath%3DGet%2520Started%7C_____0) that uses your new custom activity!
