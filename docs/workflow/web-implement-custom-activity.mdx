Implementing a custom [activity](concepts-workflow-activities-form-elements#activities) allows you to a build a reusable, completely custom behaviour to augment the [existing activities](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/activity-reference.htm%3FTocPath%3DActivities%7CActivity%2520Reference%7C_____0) that come with Geocortex Workflow. For workflows targeting Web Applications like Geocortex Viewer for HTML or Web App Builder, activities are implemented in Typescript. For workflows targeting Geocortex Mobile, or for activities used in [Server Workflows](TODO - some sort of link here...) (which can be [run using the "RunWorkflow" activity on any platform](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/server-workflows.htm#configure-client-workflow). activities are implemented using csharp and the .NET platform).

## Set up Activity Skeleton

To create a new activity:

1. Open the workflow activity SDK in Visual Studio Code
1. Run `npm run activity` in the terminal.
1. When prompted enter the name of the activity you would like to create and press `Enter`. For example, `MyActivity`.

:::tip
It's convention to use PascalCase for activity names.
You can add many activities to the same project.
:::

1. The script will perform the following operations:
    1. Create a new activity .ts file with the provided name in the `src/activities` folder.
    1. Populate the activity .ts file from an activity template.
    1. Register the activity in `src/index.ts`.
1. Open the newly created `src/activities/MyActivity.ts` file.

## Set up Activity Skeleton

```csharp
using Geocortex.Workflow.Runtime;
using System.Collections.Generic;
using System.Threading.Tasks;

namespace App1.Samples.Workflow.CustomActivity
{
    public class CustomActivity : IActivityHandler
    {
        public const string Action = "custom:wf:app::Hello";

        public Task<IDictionary<string, object>> Execute(IDictionary<string, object> inputs, IActivityContext context)
        {
            string name = "Custom Workflow!";
            if (inputs.TryGetValue("name", out var temp))
            {
                name = temp as string ?? name;
            }

            IDictionary<string, object> result = new Dictionary<string, object>()
            {
                ["hello"] = $"Hello, '{name}'"
            };

            return Task.FromResult(result);
        }
    }
}
```
