The Geocortex Mobile SDK is used to extend Geocortex Workflow functionality for Geocortex Mobile. Follow the instructions in the [Quickstart](../mobile/sdk-quick-start.mdx) project to get set up for Geocortex Mobile + Geocortex Workflow custom development.

:::important
Creating custom workflow activities for Geocortex Mobile requires the development and deployment of a custom Geocortex Mobile application.
:::
