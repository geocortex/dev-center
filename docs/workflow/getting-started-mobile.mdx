The Geocortex Mobile SDK can also be used to extend Geocortex Workflow functionality for Geocortex Mobile. Follow the instructions in the [quick start](../mobile/quick-start.mdx) project to get setup for Geocortex Mobile + Geocortex Workflow custom development.

## Development Enviroment

Visual Studio 2019 is the only officially supported development enviroment for extending Geocortex Workflow Server.
