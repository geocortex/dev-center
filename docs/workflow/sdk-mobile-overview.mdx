The VertiGIS Studio Mobile SDK is used to extend VertiGIS Studio Workflow functionality for VertiGIS Studio Mobile. Follow the instructions in the [Quickstart](../mobile/sdk-overview.mdx) project to get set up for VertiGIS Studio Mobile + VertiGIS Studio Workflow custom development.

:::important
Creating custom workflow activities for VertiGIS Studio Mobile requires the development and deployment of a custom VertiGIS Studio Mobile application.
:::

## SDK Samples

<SdkLimitationsWarning />

The VertiGIS Studio Mobile SDK includes various [SDK Samples](../mobile/sdk-samples.mdx) for development, including [a sample which demonstrate custom workflow functionality](https://github.com/geocortex/vertigis-mobile-samples/tree/master/Geocortex.Mobile.Samples/Geocortex.Mobile.Samples/Samples/Workflow/CustomActivity) .
