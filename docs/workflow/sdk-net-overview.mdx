VertiGIS Studio Workflow has a .NET runtime for executing workflows in a .NET environment. [VertiGIS Studio Mobile](https://docs.vertigisstudio.com/mobileviewer/) and [VertiGIS Studio Workflow Server](https://docs.vertigisstudio.com/workflow/latest/help/Default.htm#wf5/help/server-workflows.htm) both run workflows in a .NET environment.

Workflows for VertiGIS Studio Mobile and VertiGIS Studio Workflow Server are still authored in the [VertiGIS Studio Workflow Designer](https://apps.vertigisstudio.com/workflow/designer/). Custom .NET workflow activities can be executed by name with the [Run Activity](https://docs.vertigisstudio.com/workflow/latest/help/Default.htm#wf5/help/activities/run-activity.htm) activity. In addition, the VertiGIS Studio Workflow Typescript SDK can be used to create [activity stubs](sdk-net-register-activities.mdx) that give a user friendly interface for custom .NET workflow activities.
