:::important
Extending Geocortex Workflow for Mobile requires development and deployment of a custom Geocortex Mobile Application using the [Geocortex Mobile SDK](../../mobile/sdk-overview.mdx)
:::

Follow the instructions in the [Geocortex Mobile SDK](../sdk-mobile-overview.mdx) page to set up the environment for extending Workflow for Geocortex Mobile.

:::note
A working knowledge of [C# and the .NET platform](https://dotnet.microsoft.com/) is recommended before extending Workflow for Geocortex Mobile
:::
