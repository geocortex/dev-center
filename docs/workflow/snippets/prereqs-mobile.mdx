:::important
Extending VertiGIS Studio Workflow for Mobile requires development and deployment of a custom VertiGIS Studio Mobile Application using the [VertiGIS Studio Mobile SDK](../../mobile/sdk-overview.mdx)
:::

Follow the instructions in the [VertiGIS Studio Mobile SDK](../sdk-mobile-overview.mdx) page to set up the environment for extending Workflow for VertiGIS Studio Mobile.

:::note
A working knowledge of [C#](https://docs.microsoft.com/en-ca/dotnet/csharp/) and [.NET Standard](https://dotnet.microsoft.com/platform/dotnet-standard) is recommended before extending Workflow for VertiGIS Studio Mobile
:::
