## Prerequisites

:::important
Extending Geocortex Workflow for Mobile requires development and deployment of a custom Geocortex Mobile Application using the [Geocortex Mobile SDK](../mobile/quick-start.mdx)
:::

Follow the instructions in the [Geocortex Mobile SDK](getting-started-mobile) article setup the enviroment for extending workflow for Geocortex Mobile.

:::note
A working knowledge of [C# and the .NET platform](https://dotnet.microsoft.com/) is reccomended before extending workflow for Geocortex Mobile
:::
