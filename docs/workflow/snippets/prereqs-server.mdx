## Prerequisites

:::important
Extending Geocortex Workflow with [server side components](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/server-workflows.htm%3FTocPath%3DServer%2520Workflows%7C_____0) requires an [on premise installation of Geocortex Workflow](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/installation.htm). You will need adminstrative access to this installation.
:::

Follow the instructions in the [Geocortex Workflow Server](../getting-started-server.mdx) article to setup the enviroment for extending workflow for Geocortex Workflow Server.

:::note
A working knowledge of [C# and the .NET platform](https://dotnet.microsoft.com/) is reccomended before extending workflow for Geocortex Workflow Server
:::
