:::important
Extending VertiGIS Studio Workflow with [server side components](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/server-workflows.htm%3FTocPath%3DServer%2520Workflows%7C_____0) requires an [on-premises installation of VertiGIS Studio Workflow](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/installation.htm). You will need administrative access to this installation.
:::

Follow the instructions in the [VertiGIS Studio Workflow Server](../sdk-server-overview.mdx) page to set up your development environment and VertiGIS Studio Workflow Server extension project.

:::note
A working knowledge of [C#](https://docs.microsoft.com/en-ca/dotnet/csharp/) and [.NET Standard](https://dotnet.microsoft.com/platform/dotnet-standard) is recommended before extending VertiGIS Studio Workflow Server
:::
