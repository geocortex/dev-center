Geocortex Mobile's appearance is built around two concepts: theme and [layout](concepts-layout.mdx). The theme controls the color scheme of the app, while the layout controls the organization of content.

### Seeding a New Theme from an Existing Theme

Two prefined themes are included with Geocortex Web, `light` and `dark`. You can select a predefined theme and accent color with [Geocortex Web Designer](https://docs.geocortex.com/webviewer/latest/admin-help/Default.htm#gwv/component-settings.htm#branding-settings%3FTocPath%3DConfiguration%7CComponent%2520Settings%7C_____4).

The branding [service](concepts-components-services.mdx#services) is responsible for the theming of the application. To select a theme and accent color from within the config, you can configure the branding service in the [app config](concepts-app-config.mdx).

### Customizing Theme Colors

Every color in a theme can be fully customized by editing the `colors` property of the the branding service configuration.

:::tip
Colors can be specified with their common name, a hex value, or a rgba value.
:::

<Tabs
  defaultValue="config"
  values={[
    { label: "App Config", value: "config" },
    { label: "UI", value: "ui" },
  ]}>
<TabItem value="config">

```json
{
    "items": [
        ...
                {
            "id": "branding",
            "$type": "branding",
            "activeTheme": "custom-vampire",
            "themes": [{
                "id": "custom-vampire",
                "accentColor": "red",
                "template": "dark",
                "colors": {
                    "emphasizedButtonBackground": "purple",
                    "primaryAccentLarge": "#00b7eb",
                    "secondaryBackground": [249, 77, 0, 255]
                }
            }]
        },
        ...
    ]
}
```

</TabItem>

### Using Theme Colors In Custom Components

<!-- TODO-API - Link to Geocortex Framework UI -->

When creating custom components, you may need to style your component. If you use Geocortex UI Elements, these elements already participate in the theme, so your element will be themed automatically. However, you can also use the various theming colors in custom css, through the `var(...)` css function.

```css
.my-element {
    background-color: var(--primaryBackground);
}
```

### Changing the Visual Layout of an Application

How elements are laid out in a Geocortex Web Application is defined by an apps layout. [This article](concepts-layout.mdx) will teach you how to configure an app's layout and how to leverage it to customize the appearance of Geocortex Web. Further styling within the layout can be done using [presentation attributes](api-layout.mdx#presentation-attributes).
