---
title: Theme Colors
description: Geocortex Web - Available theme colors reference
---

Following is the interface that declares every possible theme color that can be used in a Geocortex Web Application.

:::tip
You can [inspect the css](https://developers.google.com/web/tools/chrome-devtools/css) of a Geocortex Web Application to determine what theme property a specific element uses.
:::
