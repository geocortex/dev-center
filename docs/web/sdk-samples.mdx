Alongside the [VertiGIS Studio Web SDK](sdk-overview.mdx), a [SDK samples repository](https://vertigis-web-samples.netlify.app) has been created which demonstrates using the SDK to extend VertiGIS Studio Web with custom components, operations, services, and more. The source code is open source and [available on GitHub](https://github.com/geocortex/vertigis-web-samples).

In addition to the SDK samples we also have open-sourced the [VertiGIS Studio Web Component Incubator](https://vertigis-web-incubator.netlify.app). The Component Incubator contains a collection of experimental components available for use in your applications, and also serves as a great reference to learn about building more complex, fully featured components using the SDK. The source code is open source and [available on GitHub](https://github.com/geocortex/vertigis-web-incubator).
