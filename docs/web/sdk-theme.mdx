Geocortex Web's appearance is built around two concepts: theme and [layout](concepts-layout.mdx). The theme controls the color scheme of the app, while the layout controls the organization of content.

The branding [service](concepts-components-services.mdx#services) is responsible for the theming of the application. To select a theme and customize colors, you can configure the branding service in the [app config](concepts-app-config.mdx).

## Seeding a New Theme from an Existing Theme

Two predefined themes are included with Geocortex Web, `light` and `dark`. You can select a predefined theme and accent color with [Geocortex Web Designer](https://docs.geocortex.com/webviewer/latest/admin-help/Default.htm#gwv/component-settings.htm#branding-settings%3FTocPath%3DConfiguration%7CComponent%2520Settings%7C_____4).

## Customizing Theme Colors

Every color in a theme can be fully customized by editing the `colors` property of the the branding service configuration.

:::tip
Colors can be specified with their common name, a hex value, or a rgba value. A full list of all the color keys available in a theme [can be found here](api-theme.mdx).
:::

<Tabs
  defaultValue="config"
  values={[
    { label: "App Config", value: "config" },
    { label: "UI", value: "ui" },
  ]}>
<TabItem value="config">

```json
{
    "schemaVersion": "1.0",
    "items": [
        ...
        {
            "$type": "branding",
            "id": "branding",
            "activeTheme": "custom-vampire",
            "themes": [{
                "id": "custom-vampire",
                "accentColor": "red",
                "template": "dark",
                "colors": {
                    "emphasizedButtonBackground": "purple",
                    "primaryAccentLarge": "#00b7eb",
                    "secondaryBackground": [249, 77, 0, 255]
                }
            }]
        },
        ...
    ]
}
```

## Using Theme Colors In Custom Components

<!-- TODO-API - Link to Geocortex Framework UI -->

When creating custom components, you may need to style your component. If you use Geocortex UI Elements, these elements already participate in the theme, so your element will be themed automatically. However, you can also use the various theming colors in custom css, through the `var(...)` css function.

:::tip
A full list of all the color keys available in a theme [can be found here](api-theme.mdx).
:::

```css
.my-element {
    background-color: var(--primaryBackground);
}
```
