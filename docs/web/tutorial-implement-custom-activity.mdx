VertiGIS Studio Web comes with bundled with [VertiGIS Studio Workflow](https://www.geocortex.com/products/geocortex-workflow/). VertiGIS Studio Workflow includes a [large suite of activities](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/activity-reference.htm%3FTocPath%3DActivities%7CActivity%2520Reference%7C_____0) to help you solve your business case. However, if VertiGIS Studio Workflow is missing functionality you need, you can implement a custom activity with the VertiGIS Studio Workflow SDK.
