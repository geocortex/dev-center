Geocortex Web comes with bundled with [Geocortex Workflow](https://www.geocortex.com/products/geocortex-workflow/). Geocortex Workflow includes a [large suite of activities](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/activity-reference.htm%3FTocPath%3DActivities%7CActivity%2520Reference%7C_____0) to help you solve your business case. However, if Geocortex Workflow is missing functionality you need, you can implement a custom activity with the Geocortex Workflow SDK.
