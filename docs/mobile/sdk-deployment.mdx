If you only need to modify the app config and layout of a Geocortex Mobile application, you can use the [Geocortex Mobile Designer](https://apps.geocortex.com/mobileviewer/designer/) interface. However, if you need to write custom code, you will need to deploy and manage a custom application.

### Application with Custom Code

The Quickstart provided is a standard Xamarin Forms application, and can be deployed in accordance to the [Xamarin Forms deployment guidelines](https://docs.microsoft.com/en-us/xamarin/xamarin-forms/deploy-test/).
