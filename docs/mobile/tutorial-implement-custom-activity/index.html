Implement a Custom Activity

Geocortex Mobile comes with bundled with Geocortex Workflow. Geocortex Workflow includes a large suite of activities to help you solve your business case. However, if Geocortex Workflow is missing functionality you need, you can implement a custom activity with the Geocortex Workflow SDK.
