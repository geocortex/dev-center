## Running a Workflow Programmatically

If you are building a custom application and need to manually configure the layout and app config to run a workflow, you can check out the [change default map click behavior](usecases-change-default-map-click-behavior.mdx) use case, or the [SDK sample of running a workflow](https://github.com/geocortex/vertigis-mobile-samples/tree/master/Geocortex.Mobile.Samples/Geocortex.Mobile.Samples/Samples/Workflow/RunWorkflow).
