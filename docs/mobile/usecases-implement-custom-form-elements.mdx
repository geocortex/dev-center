Geocortex Mobile comes with bundled with [Geocortex Workflow](https://www.geocortex.com/products/geocortex-workflow/). Geocortex Workflow includes a [large suite of form elements](https://docs.geocortex.com/workflow/latest/help/Default.htm#wf5/help/form-elements/auto-complete.htm%3FTocPath%3DForms%7CForm%2520Element%2520Reference%7C_____0) to help you [create custom forms](usecases-display-custom-form.mdx) to solve your business case. However, if Geocortex Workflow is missing a form element you need, you can implement a custom form element with the Geocortex Workflow SDK.
