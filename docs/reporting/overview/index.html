Overview

Geocortex Reporting is a flexible tool for creating sophisticated, professional-looking reports that include spatial and non-spatial data from ArcGIS map services, tables, databases, and other data sources. Reports can be created in all shapes and sizes, with varying complexities and requirements, from basic mailing label reports to inspection compliance reports for governing bodies like FEMA. Geocortex Reporting is part of the Geocortex suite of products.

Reports can be shared with end users allowing them to create on-demand, print-ready PDF reports about maps and features, using data from the user's current session. Reports can be run in Geocortex Web, Geocortex Viewer for HTML5, Web AppBuilder for ArcGIS, via Geocortex Workflow or directly through our convenient REST API.

Have you explored the Geocortex Report Designer?

The Geocortex Report Designer has an extensive suite of tools for laying out reports, specifying the data that you want to include, and presenting the data using charts, tables, and other configurable elements.
